# Proyecto: Informe de Práctica

Este directorio contiene los archivos fuente LaTeX del informe.
Se ha dividido en múltiples directorios para mejor organización.

- `main.tex`: Documento principal que compila todo.
- `chapters/`: Capítulos del informe.
- `styles/`: Estilos y definiciones, incluyendo portada.
- `tables/`: Tablas separadas para facilitar su edición.
- `figures/`: Figuras y gráficos.
- `appendices/`: Apéndices.
- `data/`: Datos y observaciones (archivos brutos).
